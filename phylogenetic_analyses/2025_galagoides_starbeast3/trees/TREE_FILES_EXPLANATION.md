# StarBeast3 Tree Files Explanation

**Date**: January 19, 2026  
**Analysis**: Galagoides multispecies coalescent phylogenetic analysis

## Two Types of Tree Files

### 1. `galago_full_MCC.tree` - Maximum Clade Credibility Tree

**What it is**: A single summary tree representing the "best" tree from the posterior distribution.

**Characteristics**:
- **Single tree**: Contains only ONE tree
- **Summary tree**: Generated from the posterior distribution using TreeAnnotator
- **Annotated**: Includes posterior probabilities, HPD intervals, and other statistics on nodes
- **File size**: ~5 KB (small, single tree)
- **Format**: NEXUS format with one annotated tree

**What it contains**:
- The tree topology with the maximum clade credibility
- Node annotations with:
  - Posterior probabilities (e.g., `posterior=1.0`, `posterior=0.9997777901227709`)
  - Height statistics (mean, median, 95% HPD intervals)
  - Branch length statistics
  - Population size estimates

**Use cases**:
- ✅ **Publication figures** - The tree to use for final visualizations
- ✅ **Interpretation** - Shows the most credible tree topology
- ✅ **FigTree** - Easy to open and visualize
- ✅ **Analysis** - Use for downstream analyses requiring a single tree

**Example node annotation**:
```
[&posterior=1.0,height_95%_HPD={0.02949710336644458,0.08800687657887865}]
```

---

### 2. `galago_full.species.trees` - Posterior Distribution of Species Trees

**What it is**: The full posterior distribution of all species trees sampled during the MCMC run.

**Characteristics**:
- **Multiple trees**: Contains **20,001 trees** (one per MCMC sample)
- **Raw output**: Direct output from StarBeast3 analysis
- **Full distribution**: Shows uncertainty in tree topology and branch lengths
- **File size**: ~8.7 MB (large, many trees)
- **Format**: NEXUS format with multiple trees

**What it contains**:
- Every tree sampled during the MCMC chain
- Each tree labeled with its state number (e.g., `STATE_0`, `STATE_10000`, `STATE_20000`)
- Population size estimates for each tree
- Branch lengths for each tree

**Use cases**:
- ✅ **DensiTree visualization** - Shows uncertainty in tree topology
- ✅ **Generate MCC tree** - Used as input to TreeAnnotator to create the MCC tree
- ✅ **Consensus trees** - Can generate majority-rule consensus trees
- ✅ **Uncertainty analysis** - Assess topological uncertainty
- ✅ **Additional summary trees** - Generate trees with different criteria

**Example tree entry**:
```
tree STATE_100000 = ((1[&pop=7.411859330727935E-4]:0.04185256197079027,...
```

## Key Differences Summary

| Aspect | MCC Tree | Posterior Trees |
|--------|----------|-----------------|
| **Number of trees** | 1 tree | 20,001 trees |
| **File size** | ~5 KB | ~8.7 MB |
| **Content** | Single summary tree | Full posterior distribution |
| **Annotations** | Rich (posterior probs, HPD intervals) | Basic (population sizes, branch lengths) |
| **Generated by** | TreeAnnotator | StarBeast3 directly |
| **Use for** | Publication, interpretation | Uncertainty analysis, generating summaries |

## How They're Related

1. **StarBeast3 runs** → Generates `galago_full.species.trees` (20,001 trees)
2. **TreeAnnotator processes** → `galago_full.species.trees` → Creates `galago_full_MCC.tree` (1 tree)

The MCC tree is **derived from** the posterior trees file.

## Workflow

```
StarBeast3 Analysis
    ↓
galago_full.species.trees (20,001 trees)
    ↓
TreeAnnotator (with burn-in)
    ↓
galago_full_MCC.tree (1 annotated tree)
    ↓
Visualization & Publication
```

## When to Use Which

### Use `galago_full_MCC.tree` when:
- Creating publication figures
- Interpreting phylogenetic relationships
- Need a single "best" tree
- Want annotated support values
- Opening in FigTree for visualization

### Use `galago_full.species.trees` when:
- Visualizing uncertainty (DensiTree)
- Generating additional summary trees
- Analyzing topological uncertainty
- Need to regenerate MCC tree with different parameters
- Want to see the full posterior distribution

## Additional Tree Files

### Gene Trees (`*_speciesclean.trees`)
- Separate posterior distributions for each locus (12S, 16S, CYTB, IRBP, vWFes28, vWFin11)
- Each contains ~20,000 gene trees
- Used for analyzing gene tree vs. species tree discordance

### `species.trees`
- Another species tree file (smaller, ~0.9 MB)
- May be from a test run or different analysis
- Check file dates to determine which is most recent

---

**Summary**: The MCC tree is a single, annotated summary tree perfect for publication. The posterior trees file contains all sampled trees and is used for uncertainty analysis and generating summary trees.
